CatCubes: A Web-Based Game

Project Overview

Welcome to my exciting project of a web-based game that is similar to CandyCrush. The theme of this game is cats.

Key Features

  • Can match 3-5 cats in a row to score points.
  • Functioning Scoreboard.

Future Improvements

  • Create updated images of cats so that they are squares.
  • Correct dropping cat theory so that there are no empty squares.
  • Create multiple levels and combo power-ups.
  • Match 4 cats in a cube for points.
